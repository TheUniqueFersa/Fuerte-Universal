Es una familia de microcontroladores, por lo tanto existen varias versiones del MC68HC11
Todas comparten el CPU, pero tienen distintos periféricos

| Microcontrolador                                  | ==vs== | Microprocesador        |
| ------------------------------------------------- | ------ | ---------------------- |
| CPU (Unidad central de procesamiento)             |        | Es tan solo un **CPU** |
| Junto con un conjunto de dispositivos periféricos |        |                        |
| Puertos paralelos                                 |        |                        |
| Puertos seriales                                  |        |                        |
| Temporizadores                                    |        |                        |
| Contadores                                        |        |                        |
| Convertidores Digital a Análogo y viceversa       |        |                        |
| Memoria RAM                                       |        |                        |
| Memoria ROM                                       |        |                        |
|                                                   |        |                        |
| *Todo contenido en un solo encapsulado chip*      |        |                        |
