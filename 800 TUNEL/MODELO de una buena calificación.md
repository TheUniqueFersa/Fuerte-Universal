
| Tipo            | Significado                                                                                                                                                             |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Aciertos        | Algo que hizo muy bien y que sobresale de los puntos que en sí se deben asignar (se aprecia que lo hizo con esmero)                                                     |
| Observaciones   | Algo que roza lo incorrecto sin serlo, no es mala práctica, pero tiene que tener una señal de advertencia para que no se transforme en un error de mala práctica o pero |
| Recomendaciones | Una mala práctica o un error que no afecta en sí el funcionamiento del programa, pero si debe corregirse para construir un **mejor programador**                        |
| Errores         | Algo que estuvo mal y que afecta negativamente el funcionamiento del programa                                                                                           |
| TRAMPA          | Se define como $N$ y son las acciones que implican una trampa, como plagio, uso de IA descarado etc                                                                     |
